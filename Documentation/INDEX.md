## ⚙️ Configuration

### Customize Refresh Rate
Edit `script.js`:
```javascript
CONFIG.refreshInterval = 5000; // 5 seconds
```

### Customize Colors
Edit `style.css`:
```css
:root {
    --primary-blue: #00d4ff;
    --accent-red: #ff4757;
    /* More colors... */
}
```

### Adjust Threat Thresholds
Edit `script.js`:
```javascript
CONFIG.threatScoreThresholds = {
    critical: 80,
    high: 60,
    medium: 40,
    low: 0
}
```

## 🆘 Troubleshooting

**Map not loading?**
- Check browser console
- Ensure internet connection (CDN access)
- Try refreshing page

**Charts not showing?**
- Clear browser cache
- Check JavaScript console
- Verify Chart.js library loads

**Styles look broken?**
- Check CSS file path
- Refresh with Ctrl+Shift+R
- Try different browser
